Hi, I'm Godlove — a Full Stack Developer and tech educator based in Douala, Cameroon. I build complete web systems from database design to polished UI, and I teach every step on my Codex AG YouTube channel.

My signature approach is UML-first development: every system I build starts with Use Case, Activity, and Sequence diagrams before any code is written. This ensures clarity, reduces bugs, and creates better architecture.

I've shipped real-world systems including a First Aid Management System (LifeSaver), a Waste Management platform, a dual-portal Career Bridge app, e-commerce, educational platforms, and more — all built with PHP & MySQL.

DIAGRAM TYPE 01
Use Case Diagrams
Identify all actors and map what each can do. Defines scope before writing any business logic.
DIAGRAM TYPE 02
Activity Diagrams
Flowchart-style maps of every decision point, parallel process, and workflow in the system.
DIAGRAM TYPE 03
Sequence Diagrams
Time-ordered message flow between components — showing exactly how the system communicates.
